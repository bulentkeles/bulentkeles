Project Overview

Public Bikes is a General Assembly Immersive UX Design Project Patterns For Kids. The app teaches kids the concept of patterns in a fun and highly interactive way. The app is built with an algorithm that allows for thousands of pattern variations. As the learner proceeds through the app the levels become more and more challenging.

The designs below are the upgraded UI and Illustrations. You can view the original build in the App Store.
